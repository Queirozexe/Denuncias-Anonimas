Plataforma acadêmica com foco em segurança, organização e responsabilidade

Sistema de denúncias anônimas com visual moderno e navegação clara

Este site foi desenvolvido para permitir que cidadãos relatem situações de risco, violência, irregularidades e problemas urbanos sem necessidade de identificação, de forma simples, estruturada e com acompanhamento por protocolo.

Registrar denúncia Consultar status
Importante: este projeto é uma simulação educacional para Github Pages. O envio de evidências, armazenamento e acompanhamento funcionam no navegador com dados locais do usuário.
100% anônimo Nenhum campo exige nome, CPF, e-mail ou qualquer dado pessoal.
Protocolo automático Cada denúncia gera um número único para acompanhamento posterior.
Status de andamento Consulte etapas como recebida, em análise, encaminhada e concluída.

Categorias de denúncia

As categorias foram organizadas em blocos para facilitar o registro correto de cada situação.

A) Segurança e Violência

  • Violência contra a mulher
  • Violência contra crianças
  • Violência contra idosos
  • Furto/Roubo
  • Drogas
  • Atividade suspeita

B) Direitos e Conduta

  • Assédio moral/sexual
  • Bullying/Cyberbullying
  • Discriminação
  • Direitos humanos
  • Denúncia institucional

C) Meio Ambiente e Bem-Estar

  • Maus-tratos a animais
  • Crimes ambientais
  • Perturbação do sossego
  • Situações de risco

D) Cidade e Serviços

  • Infraestrutura urbana
  • Serviços públicos
  • Transporte e trânsito
  • Irregularidades escolares

Estrutura pensada para Github Pages

Projeto estático em HTML, CSS e JavaScript puro, fácil de publicar em repositórios acadêmicos.

Foco em design

Layout responsivo, tipografia forte, blocos bem definidos e hierarquia visual clara para melhor usabilidade.
